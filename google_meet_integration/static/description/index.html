Overview

With this module user can easily schedule the event(meeting) time, date and hours from Odoo and the attendees can directly join meeting from the Google Meet URL generated in Odoo.

Features

Set up your Google Meet schedule in Odoo itself.

Generate Google Meet link from Odoo.

Users can join the Meeting invitation from their Google Calendar as well.

Setup & Configuration

Prerequisites

  • Required Apps

    Calendar

Configuration of Google Developer Console

  • Create project on Google cloud platform and generate client id, secret id, and enable Google Calendar API.

  • Copy to the below Link for open the tutorial.

COPY LINK
https://bit.ly/3vtkx4I

Configuration

Company Configuration 1-Step:

  • Click on the Action button and select Recent Logs from the dropdown menu to view the recent activities performed by the userAdd your Credentials to client id and client secret for Google API configs.
  • This is one time configuration which is done at the company level.

Generate Token on User 2-Step:

  • To establish connection between Odoo and Google API, under Calendar Meetings, user needs to generate the token using the Generate Token button. Every user has to generate their own token by going to the User Preferences.
  • This will link the Users Google ID with Odoo.
  • As soon as user clicks onGenerate Token, Refresh token and Validity appears which is not further editable.

Screenshots

Meeting Generate Process:

  • Now, user can schedule the event(meeting) by filling up details like- name, start time, duration (in hrs or whole day), location, tags, and reminders.

Enable Generate Meet Link:

  • Upon enabling Generate Google Meet Link checkbox, a meeting URL is directly sent to attendees inbox.

User can also share the link with others.

Google Calender Meeting Link:

  • User can click more detail option to redirect in Google Calendar.
  • Attendees and the user can directly join the event/meeting from their Google Calendar itself.
